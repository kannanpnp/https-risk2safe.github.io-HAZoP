# Risk2Safe BowTie Studio

AI-assisted bow tie risk assessment and Safety Critical Task Analysis (SCTA) for process safety professionals.

🌐 **Live application:** [Open Risk2Safe BowTie Studio](./index.html)

📋 **Manual version (no AI):** [Open Manual BowTie](./manual.html)

---

## What it does

- Build visual bow tie diagrams with full editing capability
- Attach P&IDs, process descriptions, and ESD cause-and-effect logic as context for AI analysis
- Auto-flag Safety Critical Elements (SCEs) using a consequence × human-involvement risk matrix
- Run deep 7-step Safety Critical Task Analysis (SCTA) on any identified SCT
- Generate regulator-ready Word and PDF reports with cover page, executive summary, methodology, and full proformas
- Export to multi-sheet Excel for spreadsheet workflows

## Using the AI features

The AI version requires an Anthropic API key. Each user provides their own — keys are stored only in your browser and sent directly to Anthropic. No backend, no intermediary.

1. Visit [console.anthropic.com](https://console.anthropic.com/settings/keys)
2. Sign up (free trial credits included)
3. Click **Create Key** and copy it
4. Paste it into the app when prompted

Typical cost per analysis: $0.01–0.15 USD depending on depth.

## Privacy

- All data stays in your browser
- Uploaded documents and API key never leave your device except to go directly to Anthropic
- No analytics, no tracking, no telemetry
- This tool supports the bow tie and SCTA process but does not replace formal HAZID/HAZOP/SCTA workshops with multidisciplinary teams
